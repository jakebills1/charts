chart management system project idea

A searchable management system for pdf's of charts and leadsheets. Documents should be found by title, artist, genre, or band associated with.

1 introduction

1.1 This would be an organizational tool for musicians to manage their many leadsheets and charts that they produce. It should eventually include the ability to make a playlist.

2 implementation

2.1 This could be a rails web app, with a react front end. It would consist of a log in screen, a dashboard for your charts, a search bar, a drag and drop upload space…

2.1.1 Documents would be stored via a cloudinary upload. Each document would belong to the user model and have the following data points: url, name, artist, group( deb, mb, thtrio,,,), genre, &c
